Question 1

Répartition des produits achetés en fontion du volume en cm3

Le but de cette question est de pouvoir déterminer la taille de l’entrepôt de stockage mais également le type de cartons qu’il faut utiliser pour l’expédition des produits que nous déciderons de vendre.

Fichier et attributs utilisés

Le fichier utilisé pour cette question est le fichier : olist_product_dataset.csv.
Les attributs utilisés pour calculer le volume des produits sont : product_length_cm, product_height_cm et product_width_cm.

Mapper

Le mapper calcul le volume de chaque produit avec les 3 attributs nécessaires. La clef est donc la fourchette de valeur à laquelle appartient chaque volume. La valeur associée a cette clef est la valeur 1.

Redcucer

Le reducer compte le nombre d'occurence de chaque clef (donc de tranche de volume). Les données sont ensuite triées afin de faciliter l'affichage dans le graphique.
