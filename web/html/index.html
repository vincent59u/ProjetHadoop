Projet de Big Data sur le E-commerce Brésilien

Matthieu VINCENT

M2 ACSI

Le but de ce projet était de traiter un grand volume de données concernant le e-commerce brésilien avec les techniques de Big Data. Les indicateurs présents sur ces pages web ont été commandés par les étudiants économistes de Nancy.

L'ensemble des données utilisées se trouvent ici : https://www.kaggle.com/olistbr/brazilian-ecommerce

Ce projet a été entièrement réalisé avec Hadoop et Java. Ce choix a été motivé par le côté verbeux d'Hadoop qui permet de bien comprendre le fonctionnement de Map Reduce. Cette étape est pour moi importante avant d'utiliser Spark.
La partie web pour présenter les résultats s'est faite en JavaScript et HTML/CSS. Les graphiques ont été faits avec les librairies chartjs et leaflet.
